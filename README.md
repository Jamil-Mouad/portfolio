## Description

A modern personal portfolio built with React 19 and TypeScript, featuring an AI-powered chat assistant that lets visitors ask questions about my skills and projects. The site combines immersive 3D visuals rendered with Spline and fluid Framer Motion animations to create a standout user experience. Every section—from project showcases to the contact form—is fully responsive and supports dark mode out of the box.

## Highlights

- **AI Chat Assistant** — Conversational interface powered by the Anthropic Claude API; answers questions about skills, projects, and availability in real time via server-sent events
- **3D Interactive Model** — Embedded Spline scene (`scene.splinecode`) that responds to user interaction directly in the browser
- **Framer Motion Animations** — Scroll-triggered and entrance animations throughout every section for a polished, professional feel
- **Responsive & Mobile-Friendly** — Tailwind CSS utility-first layout that adapts seamlessly from mobile to ultra-wide screens
- **Dark Mode** — System-preference-aware dark/light theme powered by Radix UI primitives
